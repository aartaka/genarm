* Getting Started
Clone the Git repository:
#+begin_src sh
  git clone --recursive https://github.com/aartaka/genarm ~/common-lisp/
#+end_src

And then load ~:genarm~ in the REPL:
#+begin_src lisp
  (asdf:load-system :genarm)
  ;; or, if you use Quicklisp
  (ql:quickload :genarm)
#+end_src
